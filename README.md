## Hvordan starte frontend

I terminalen:

- cd til "frontend"
- Kjør `npm install`
- Kjør `npm start`

## Hvordan starte backend

### Første gang:

I terminalen:

- Kjør `pip install pipenv`
- Kjør `pipenv shell`
- Kjør `pipenv install`
- cd til "backend"
- Kjør `python manage.py migrate`
- Kjør `python manage.py runserver`

### Ellers:

- Kjør `pipenv shell`
- Kjør `python manage.py runserver`
